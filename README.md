# 🎭 Mood UI Generator

**Mood UI Generator** is an experimental UI design tool that explores a simple idea:

> *What would an interface look like if it had a mood?*

Instead of designing static components, this tool lets you generate expressive UI systems driven by emotion — from **sad and calm** to **chaotic and energized**.

👉 **Live Demo:**  
https://laurandreea10.github.io/Mood-UI-Generator/

---

## ✨ Overview

Mood UI Generator transforms emotional states into visual systems by dynamically adjusting:

- color palettes & gradients
- typography & spacing
- layout & structure
- shadows, borders & glass effects
- animations & motion behavior

It’s not just a style generator — it’s a **visual experimentation playground** for expressive UI.

---

## 🧠 Core Idea

Traditional UI design focuses on usability and consistency.

This project explores:
- emotional UI
- visual storytelling through interface
- mood-driven design systems

Each mood influences multiple layers of the UI simultaneously, creating cohesive but highly distinct visual outputs.

## 🎛 Features

### 🎭 Mood-based UI generation
Switch between predefined moods:
- **Sad**
- **Energized**
- **Chaotic**
- **Calm**
- **Dreamy**

Each mood instantly transforms the entire UI.

Fine-grained control over visuals:

- Style mode: `Solid / Gradient`

  - Conic
- Up to **4 color stops**
- Gradient angle / radial position
- Opacity

- Glass intensity
- Noise texture
- Background image upload
